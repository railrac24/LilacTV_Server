Activation

처음 설치 후 라일락TV 활성화(Activation)에 대해 알아봅니다. 처음 설치시 Setup 화면에 대해 살펴봅니다. [설정]메뉴에서 와이파이 설정에 대해서도 설명합니다.

기본 메뉴 설명 #1

기본 메뉴중 [검색] / [영화] 메뉴의 기능을 살펴보고 "즐겨찾기" 기능에 대해 자세히 알아봅니다. 또한 리모트 컨트롤러의 각 버튼의 기능과 실제 사용방법에 대해 알아봅나다.

기본 메뉴 설명 #2

기본 메뉴중 [드라마] / [예능 VOD] / [시사다큐] / [애드온] / [YouTube] / [TV 시청] / [설정] 메뉴의 기능을 살펴봅니다. 특히 [TV 시청]메뉴에 대해 자세히 알아봅니다.

시스템 재설치

라일락TV 사용중에 예기치 못한 오류로 인해 소프트웨어 재설치가 필요한 상황에 대해 알아봅니다. 라일락TV 자체에서 [설정] > [시스템]메뉴를 이용하는 방법과 "라일락TV" 공식사이트에서 펌웨어를 다운받아 설치하는 방법에 대해서 알아봅니다.
